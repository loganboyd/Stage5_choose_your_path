iOS Dev Course Structure Notes

I choose to do the Intro to iOS App Development with Swift. In this course we used Xcode to develop an iOS iPhone application that records sound, can add a filter, and plays back sound.
 
  Lesson one: I learned about Xcode 
  Xcode is an iOS developer application built by Apple. Xcode is a combined GUI and text editor that makes writing and visualizing code more simple. There are 3 main sections in Xcode: Navigator, Editor, and Utility.  
 
  Lesson two: I learned how to add and correctly size a button. 
 
  Lesson three: I learned how to play audio, navigate screens and change the audio speed.  
 
  Lesson four: In lesson four we completed the application by learning how to record and create an audio file. We also learned how to manipulate the audio file by changing the pitch and speed of the file.
